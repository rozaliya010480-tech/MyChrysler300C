PWA «Мой Chrysler 300C» — версия v10

Добавлено:
- Экспорт резервной копии в JSON-файл.
- Импорт резервной копии из JSON-файла.
- Кнопка сброса данных приложения.
- Все основные функции v9 сохранены.

Как пользоваться:
1. Открой «Главная».
2. В блоке «Резервная копия» нажми «Экспорт резервной копии».
3. Сохрани файл в надёжное место.
4. Для восстановления нажми «Импорт резервной копии» и выбери сохранённый JSON-файл.

Важно:
Если была установлена старая версия, удали старую иконку приложения и установи новую заново.
