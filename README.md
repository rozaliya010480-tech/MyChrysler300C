# Мой Chrysler 300C — APK с уведомлениями через GitHub Actions

Это Android-проект для сборки APK через GitHub.

## Что добавлено в этой версии

- Android-уведомления.
- Запрос разрешения на уведомления на Android 13+.
- Фоновое напоминание каждые 12 часов через WorkManager.
- Кнопка проверки уведомления внутри приложения.
- WebView-приложение с журналом Chrysler 300C.
- Экспорт/импорт резервной копии.
- Иконка Chrysler 300C.

## Как собрать APK через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы из этой папки в репозиторий.
3. Открой вкладку **Actions**.
4. Выбери **Build Android APK**.
5. Нажми **Run workflow**.
6. После завершения сборки открой запуск.
7. Внизу в **Artifacts** скачай **MyChrysler300C-debug-apk**.
8. Внутри будет `app-debug.apk`.

## Важно

- При первом запуске Android спросит разрешение на уведомления. Нужно нажать **Разрешить**.
- Фоновое уведомление настроено каждые 12 часов.
- В приложении на главной странице есть кнопка **Проверить уведомление**.
- Это debug APK, Android может предупредить об установке из неизвестного источника.
